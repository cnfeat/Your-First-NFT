但其实

- 钱包没有钱，就没有风险

## 源起


我进入 Web3，纯粹是年度关键词[「好奇」](http://newsletter.hardwaylab.com/issues/2022-973735)的驱动。
​

刚进入 Web3 ，我对它第一印象就是，做什么都需要钱，买个币要钱，买个图片要钱，免费的也要手续费……
​

当时就在想，能不能让我有动机先去注册一个钱包，让我有理由去试一试，去看一看，然后再考虑下一步怎么走？
​

后来接触到了 NFT，发现到处谈论的是白名单。
​

怎么去获取白名单呢？
​

你要满足大量的条件，例如社区在线聊天时长等级、邀请社区人头、混社区看一波又一波的通告、参加解密游戏、参加项目创意比赛、去学英语学画画、学做视频、学写项目故事、撰写精美简历、申请成为社区管理员等。
​

**这种为获取白名单而付出的大量的时间、精力、金钱的升级内卷的过程，戏称为「肝白」。**
**​**

肝白对于 Web3 新手，也是有好处的。
​

你可以深入了解 NFT 项目起源故事，创始团队的初心及运营能力，白名单的配置是否合理，遇到各种玩家的冲突如何解决。你也可以熟悉加密市场是怎样运作的，项目方哪些动作是对用户有价值的，哪些人真正赚到了钱，知道如何判断加密项目的真正价值。
​

但即使肝到了白，你还是要买，你只是获得某个 NFT 项目的优先购买权而已。


回头一想，我是 Web3 新手呀，Web3 规则啥的我都还不懂，为什么一开始就要去肝白？为什么我一开始就要冒着风险去交易？


能不能再简单一点？有没有免费的？
​

有的，空投。
​

空投，在《绝地求生》游戏中，是给面向所有参加玩家，空中投放高级枪支道具，鼓励玩家争抢，激发活跃，加快游戏进度。


多数 NFT 项目也是类似，只不过他们空投对象是加密圈的活跃者，你理解成为「托」也好，「 KOL」也好，「投机者」也好，总之，获得空投的人，成为利益维护者，社区建造者，为项目方摇旗呐喊，  提高 NFT 的价格，吸引更多人过来过来购买。


一般来讲，Web3 新手是没有空投资格的，但事实上，Web3 新手是可以拥有空投资格的。
​

只不过这件事太简单了，太违背那些 Web3 老手的常识——这种吃力不赚钱的事情，谁去做呀。
​

但，又有何不可呢？
​

于是，我就发起了 Your First NFT 项目，算是回顾自己当初进入 Web3 困惑的回响行动。
​

花点时间设计了一张 NFT，直接铸造 10000 张，免费手工空投给所有 Web3 人。
​

当我想做一件事，就是要用最笨的方法，也要去完成它。
